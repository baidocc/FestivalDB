# Festival Database System

This project is designed to manage and organize data for a music festival efficiently. It focuses on handling information related to festivals, concerts, artists, clients, tickets, and employees using **SQL** and **PL/SQL** in an Oracle environment.

---

## About the Project

The **Festival Database System** provides a complete relational data model for a multi-day music festival.  
It allows the management of concerts, performing artists, ticket sales, clients, and employees involved in the event.  
The project emphasizes data integrity through relationships, normalization, and constraints while demonstrating the use of advanced SQL and PL/SQL operations such as subqueries, functions, and conditional expressions.

---

## Features

**Festival Management** – Store and organize details about each festival, including name, location, and schedule.  
**Concert Scheduling** – Manage concerts, artists, and the stages they perform on.  
**Ticket Management** – Handle multiple ticket types (Standard, VIP, Gold) using an ISA hierarchy structure.  
**Client Records** – Register and track client information and their purchased tickets.  
**Employee Assignment** – Manage employees assigned to various concerts and roles.  
**Advanced SQL Queries** – Includes five complex queries using subqueries, aggregate functions, date/string functions, CASE expressions, and CTEs (WITH clause).  
**Data Operations** – Demonstrates INSERT, UPDATE, and DELETE operations for maintaining data consistency.
